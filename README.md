# SportBrands_FinancialAnalysis (2021–2025)
ACC102 Track 2 - GitHub Data Product

---

## 1. Project Overview
This project analyzes and compares the financial performance of Nike and Adidas from 2021 to 2025 using key financial indicators: Revenue, Earnings Per Share (EPS), and Return on Equity (ROE).

The goal is to identify trends, compare operational stability, and draw conclusions about the two companies’ financial health, targeting investors, business students, and financial analysts.

---

## 2. Dataset Information
- **Data source**: Nike and Adidas annual reports (2021–2025)（https://investors.nike.com/）and（https://www.adidas-group.com/en/investors/）
- **Access date**: 24th April 2026
- **Location**: All data files are stored in the `data/` directory
- **Key metrics**:
  - Revenue (Million USD)
  - Earnings Per Share (EPS, USD)
  - Return on Equity (ROE, %)

---

## 3. Python Methods Used
- **pandas**: Data loading, cleaning, and preview
- **matplotlib**: Data visualization and comparison charts
- **Basic descriptive analysis**: Identifying trends and performance differences

---

## 4. Key Findings
1. **Revenue**: Nike maintained significantly higher revenue than Adidas across all five years.
2. **EPS Stability**: Nike’s EPS remained stable, while Adidas experienced large fluctuations, including a negative EPS in 2023.
3. **ROE Performance**: Nike delivered consistently higher and more stable ROE, indicating stronger returns for shareholders.
4. **Recovery Trend**: Adidas showed signs of recovery in 2024–2025 after its 2023 operational challenges.
